⚠️ IMPORTANT: Chrome PWA Installation Requirements

Chrome requires specific conditions for the "Install app" button to appear:

✅ REQUIRED CONDITIONS:
1. manifest.json file MUST be present and in same folder
2. manifest.json must have CORRECT configuration
3. HTTPS OR localhost (not file:// protocol)
4. Manifest must link to valid icons
5. Service worker must be registered
6. Standalone display mode required

⚠️ IF YOU'RE OPENING FROM FILE (file:// protocol):
   The install button won't appear in standard Chrome!
   Solution: See "LOCAL TESTING SETUP" below

🔧 SOLUTION 1: LOCAL WEB SERVER (RECOMMENDED - 2 MINUTES)

This is the PROPER way to test PWA installations locally.

WINDOWS:
════════
1. Download Python 3: https://python.org
2. Install it (check "Add Python to PATH")
3. Open Command Prompt (Win+R, type "cmd", press Enter)
4. Navigate to your folder:
   cd C:\Users\YourUsername\Desktop  (or wherever you put the files)
5. Start Python server:
   python -m http.server 8000
6. Open browser:
   http://localhost:8000
7. Click SICE_SOC_MAIN_PWA.html
8. NOW "Install app" button appears! ✅
9. Press Ctrl+Shift+Del → Delete browsing data (clear cache)
10. Reload page
11. Wait 3 seconds, then look at address bar
12. "Install SICE SOC" button appears!
13. Click Install ✅

MAC:
════
1. Open Terminal (Cmd+Space, type "terminal")
2. Navigate to your folder:
   cd ~/Desktop  (or wherever)
3. Start Python server:
   python3 -m http.server 8000
4. Open browser:
   http://localhost:8000
5. Click SICE_SOC_MAIN_PWA.html
6. "Install app" button should appear ✅
7. Click it to install!

LINUX:
══════
1. Open Terminal
2. Navigate to folder:
   cd ~/Desktop
3. Start Python server:
   python3 -m http.server 8000
4. Open browser:
   http://localhost:8000
5. Click HTML file
6. Install button appears!

🔧 SOLUTION 3: CHROME DEVTOOLS METHOD (ADVANCED)

Force Chrome to recognize PWA locally:

1. Open SICE_SOC_MAIN_PWA.html in Chrome
2. Press F12 (open DevTools)
3. Go to "Application" tab
4. Left menu → "Manifest"
5. Check if manifest.json loads (should show green checkmark)
6. If not:
   ├─ Go to "Console" tab
   ├─ Look for red errors
   └─ Fix any CORS or path issues
7. Go to "Service Workers" tab
8. Check if service worker is registered
9. Check "Update on reload" checkbox
10. Reload page (F5)
11. Now try to install

❓ TROUBLESHOOTING:

Q: I still don't see the Install button
A: 1. Make sure BOTH files are in same folder
   2. Check that manifest.json exists and is valid
   3. Use a local web server (not file://)
   4. Clear browser cache (Ctrl+Shift+Del)
   5. Reload page
   6. Wait 3-5 seconds
   7. Try F12 → Application → Manifest (check for errors)

Q: DevTools shows manifest error
A: 1. Check manifest.json is in same folder as HTML
   2. Verify it's named EXACTLY "manifest.json" (lowercase)
   3. Check for JSON syntax errors
   4. Make sure path in HTML is correct: href="manifest.json"

Q: Service Worker won't register
A: 1. This requires HTTPS or localhost
   2. Use Solution 1 (Python server) for localhost
   3. Check browser console (F12) for errors

Q: "Install" button appears but clicking does nothing
A: 1. Refresh page (F5)
   2. Clear cache (Ctrl+Shift+Del)
   3. Try a different browser (Chrome recommended)
   4. Check DevTools → Application tab for issues

Q: App installed but won't open
A: 1. Right-click icon → Properties
   2. Check shortcut target path
   3. Make sure HTML file still exists
   4. Uninstall and reinstall
