❓ FREQUENTLY ASKED QUESTIONS:

Q: Do I need OPS_.html or other files?
A: NO! Everything is embedded in one file. No dependencies.

Q: What if I can't find the "Install app" button?
A: It appears in Chrome's address bar (top right). 
   If not visible, try: 3-dot menu → "Install app"

Q: Does it work offline?
A: Yes! Service worker caches the app on first load.
   Works completely offline after that.

Q: Where does my data go?
A: All stored locally in browser (localStorage).
   Never sent to any server.
   100% private, 100% on your machine.

Q: Can I use this on my phone?
A: Yes! Works perfectly on iPhone, Android, mobile browsers.

Q: How do I uninstall the app?
A: Right-click icon → Uninstall (Windows)
   Or remove from home screen (mobile)

Q: What if something goes wrong?
A: Refresh page (F5)
   Clear browser data and reload
   Reinstall the app

Q: Can I print the reports?
A: Yes! Press Ctrl+P (or Cmd+P) while viewing report
   Save as PDF from print dialog

🔧 TROUBLESHOOTING:

Problem: "File not found" error
Solution: Make sure manifest.json is in same folder
          Both files should be together

Problem: "Install app" button doesn't appear
Solution: 1. Try opening in Chrome (not Firefox, Edge, etc.)
          2. Refresh page (F5)
          3. Check menu (3 dots → Install app)

Problem: Reports don't show when I click
Solution: 1. Make sure you're logged in
          2. Refresh page (F5)
          3. Try again

Problem: WhatsApp copy doesn't work
Solution: 1. Try copying manually (right-click → Copy)
          2. Or use Export PDF instead
          3. Print to PDF for archival

Problem: vCard won't import
Solution: 1. Make sure file is .vcf
          2. Email to yourself first
          3. Try different email app
          4. Check your phone's contacts settings
